Maribo Bryghus is a Danish Brewery located in the town of Maribo. The Brewery was founded in 1895 by Christian Jørgensen as Thor Brewery (Thor Bryggeri). It was renamed ten years later to avoid confusion with the Thor Brewery in Randers. In 1997 Maribo Bryghus was acquired by Albani Brewery, which later merged with Bryggerigruppen, now Royal Unibrew.
