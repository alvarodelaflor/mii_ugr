STacy

From Wikipedia, the free encyclopedia

The STacy was a portable version of the Atari ST. Originally designed to operate on 12 standard C cell flashlight batteries for portability, when Atari finally realized how quickly the machine would use up a set of batteries (especially when rechargeable batteries of the time supplied insufficient power compared to the intended alkalines), they simply glued the lid of the battery compartment shut, and soon discontinued the machine.

In its features comparable to Macintosh Portable.

[edit] OS

The STacy is shipped with TOS 1.04.

[edit] Specifications

  • Processor: Motorola 68HC000, running at 8 MHz
  • RAM: 1 MiB onboard, can be expanded to 4 MiB
  • ROM: 192 KiB
  • Blitter
  • Real-Time Clock Lithium Battery
  • Hard disk: ACSI-SCSI 20-40 MB
  • Floppy disk
  • Systems supported: TOS 1.04
  • Parallel: Yes (1 port)
  • Serial: Yes (1 port)
  • ACSI, ACSI Internal (same as in Mega STE)
  • FDD: Yes (1 port)
  • Midi: Yes (2 ports)
  • External keyboard
  • Modem: optional (used for this model's expansion port)
  • TrackBall
  • Screen: 10,4" EPSON LCD passive matrix Backlight
