Ōzu Domain (大洲藩, Ōzu-han?) was a Japanese han or fief centered on Ōzu in Iyo Province (now Ehime Prefecture). It was governed from Ōzu Castle.

Ōzu Domain was abolished, along with all the other han, in 1871.
