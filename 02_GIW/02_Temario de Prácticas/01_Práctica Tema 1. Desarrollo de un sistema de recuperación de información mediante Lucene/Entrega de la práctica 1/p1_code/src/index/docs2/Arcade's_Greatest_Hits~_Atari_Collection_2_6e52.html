'Arcade's Greatest Hits: The Atari Collection 2' is a compilation of six Midway arcade games: Crystal Castles, Gauntlet, Marble Madness, Millipede, Paperboy, and RoadBlasters. The compilation contains artwork and info on each game, and all games are presented in their original format.

[edit] Trivia

  • Gauntlet, Marble Madness, Paperboy, and RoadBlasters can be found on Midway Arcade Treasures for Playstation 2, Xbox, GameCube, and PC. Crystal Castles and Millipede can be found on Atari Anthology for Playstation 2, Xbox, and PC.
  • The menu for the compilation is very similar in theme to Marble Madness.
