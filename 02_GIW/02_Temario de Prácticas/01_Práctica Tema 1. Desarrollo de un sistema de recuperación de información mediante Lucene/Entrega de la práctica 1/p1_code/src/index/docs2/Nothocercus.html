Nothocercus is a genus of birds in the tinamou family. This genus comprises three members of this South American family.

The species are:

  • Highland Tinamou, Nothocercus bonapartei
  • Tawny-breasted Tinamou, Nothocercus julius
  • Hooded Tinamou, Nothocercus nigrocapillus
