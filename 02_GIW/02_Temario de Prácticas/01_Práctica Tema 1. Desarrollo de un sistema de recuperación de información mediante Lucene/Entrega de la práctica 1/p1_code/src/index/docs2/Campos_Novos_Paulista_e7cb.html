Campos Novos Paulista

From Wikipedia, the free encyclopedia

Coordinates: 22°36′11″S, 50°00′09″W Campos Novos Paulista is a municipality in the state of São Paulo in Brazil. The population in 2003 is 4,266 and the area is 485.87 km². The elevation is 446 m.
