Soisy-sur-École is a village and a commune in the Essonne département, in the French region of Île-de-France.

[edit] Demographics

At the 1999 census, the population was 1,321. The estimate for 2006 was 1,358.

Inhabitants of Soisy-sur-École are known as Soiséens.
