Phillippe Édouard Léon van Tieghem

From Wikipedia, the free encyclopedia

Phillippe Édouard Léon van Tieghem (1839-1914) was a French botanist.

The standard author abbreviation Tiegh. is used to indicate this individual as the author when citing a botanical name.[1]
