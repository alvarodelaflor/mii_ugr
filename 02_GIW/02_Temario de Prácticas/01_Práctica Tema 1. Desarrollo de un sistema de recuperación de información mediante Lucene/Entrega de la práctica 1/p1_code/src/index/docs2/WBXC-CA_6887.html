WBXC-CA is the America One affiliate for Champaign, Illinois. It is owned by L4 Media Group, and broadcasts on UHF channel 46. Like most over-the-air MTV2 affiliates, it was an affiliate of The Box until that network's acquisition by Viacom in 2001.
