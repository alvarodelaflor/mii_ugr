Bob Hoffmeyer (born July 27, 1955 in Dodsland, Saskatchewan) is a retired Canadian professional ice hockey player. He played in the National Hockey League (NHL) for the Chicago Black Hawks, Philadelphia Flyers, and New Jersey Devils.
