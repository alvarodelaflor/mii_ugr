Miss World Sports

From Wikipedia, the free encyclopedia

Miss World Sports, Miss World Sport, or Miss Sport is a title and award given to the winner of a sports event at Miss World. It is a fast track or preliminary event, giving the winner a automatically spot in the semifinals. So far there are 3 Miss World Sports winners starting at 2003. In 2005, there was no Miss Sport winner because there it wasn't a fast track event. Starting in 2006, the competition was back again.

[edit] Rules

All the contestants have to be trained in sit ups, push ups, jumping jacks, and etc. The fastest women get to move on to the next round. From there on they get tested on the sports (swimming, long jump, etc). Then the top 3 winners are announced.
