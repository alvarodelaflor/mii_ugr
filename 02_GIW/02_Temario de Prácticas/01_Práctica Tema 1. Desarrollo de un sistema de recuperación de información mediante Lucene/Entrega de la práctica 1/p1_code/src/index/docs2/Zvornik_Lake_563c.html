Zvornik Lake

From Wikipedia, the free encyclopedia

Zvornik Lake
Location Mali Zvornik, Mačva District
Lake type reservoir
Basin countries Serbia

Zvornik lake (Zvorničko jezero) is an artificial lake in Mali Zvornik, Mačva District, Serbia. It was issued after building a hydroelectric power station in 1954.
