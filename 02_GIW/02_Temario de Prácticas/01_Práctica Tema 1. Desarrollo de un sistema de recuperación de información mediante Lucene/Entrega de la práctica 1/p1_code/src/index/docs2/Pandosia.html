Pandosia

From Wikipedia, the free encyclopedia

Pandosia was the name of several ancient cities, including:

  • Pandosia (Epirus), in Epirus, Greece
  • Pandosia (Bruttium), in Bruttium (now Castrolibero,now Acri in Calabria), Italy
  • Pandosia (Lucania), in Lucania (now Basilicata), Italy
