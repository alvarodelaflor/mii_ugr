The Painted Batagur, Painted Terrapin, Saw-jawed Turtle, or Three-striped Batagur (Callagur borneoensis) is a species of turtle in the Bataguridae family. It is monotypic within the genus Callagur. It is found in Brunei, Indonesia, Malaysia, and Thailand.
