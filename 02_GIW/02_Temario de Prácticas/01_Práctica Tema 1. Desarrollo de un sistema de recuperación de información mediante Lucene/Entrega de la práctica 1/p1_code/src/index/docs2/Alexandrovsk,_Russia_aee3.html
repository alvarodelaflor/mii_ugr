Alexandrovsk, Russia

Alexandrovsk (Russian: Алекса́ндровск) is a town in Perm Krai, Russia, situated some 185 kilometers (115 mi) north-east of Perm on the Lytva River (Kama's tributary). It was founded in 1783. In 1929, it was granted urban-type settlement status and in 1951—town status. Population: 16,231 (2002 Census);[1] 20,073 (1989 Census).[2]
