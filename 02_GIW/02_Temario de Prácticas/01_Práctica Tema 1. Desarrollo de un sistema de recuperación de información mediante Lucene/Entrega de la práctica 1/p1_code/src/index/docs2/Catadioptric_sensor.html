Catadioptric sensor

From Wikipedia, the free encyclopedia

A catadioptric sensor is a visual sensor that contains mirrors (catoptrics) and lenses (dioptrics). The word catadioptric originally appeared in telescope design, but the term catadioptric sensor has come to refer to panoramic sensors created by pointing a camera at a curved mirror.
