Yuliya Gippenreyter

From Wikipedia, the free encyclopedia

Yuliya Borisovna Gippenreyter, Russian: Ю́лия Бори́совна Гиппенре́йтер (born 25 March 1930,Moscow) is a modern Russian psychologist. Her researches are dedicated to experimental psychology (sight, perception, attention, motion), family therapy, neuro-linguistic programing etc.
