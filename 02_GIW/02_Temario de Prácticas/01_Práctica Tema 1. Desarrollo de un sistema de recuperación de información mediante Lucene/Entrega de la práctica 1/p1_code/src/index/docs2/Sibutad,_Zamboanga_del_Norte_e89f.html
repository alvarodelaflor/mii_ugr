Sibutad, Zamboanga del Norte

From Wikipedia, the free encyclopedia

Sibutad is a 4th class municipality in the province of Zamboanga del Norte, Philippines. According to the 2000 census, it has a population of 15,635 people in 3,444 households.

[edit] Barangays

Sibutad is politically subdivided into 16 barangays.

  • Bagacay
  • Calilic
  • Calube
  • Delapa
  • Kanim
  • Libay
  • Magsaysay
  • Marapong
  • Minlasag
  • Oyan
  • Panganuran
  • Poblacion (Sibutad)
  • Sawang
  • Sibuloc
  • Sinipay
  • Sipaloc
