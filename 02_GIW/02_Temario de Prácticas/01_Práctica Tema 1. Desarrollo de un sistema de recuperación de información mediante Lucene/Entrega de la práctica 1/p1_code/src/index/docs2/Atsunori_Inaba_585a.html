Atsunori Inaba

From Wikipedia, the free encyclopedia

Atsunori Inaba (稲葉 篤紀, born August 3, 1972 in Aichi, Japan) is a professional baseball player for the Hokkaido Nippon Ham Fighters in Japan's Nippon Professional Baseball. He was the Most Valuable Player of the 2006 Japan Series.
