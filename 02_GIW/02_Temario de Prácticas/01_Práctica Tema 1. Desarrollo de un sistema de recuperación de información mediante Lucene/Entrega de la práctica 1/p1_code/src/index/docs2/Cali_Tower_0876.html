Cali Tower

From Wikipedia, the free encyclopedia

"Torre de Cali" ( "Cali Tower ) is a 46-story skyscraper in downtown Cali, Colombia. The tower has a full height of 205 meters if the communication masts atop are counted. However, roof height is 183.5 meters, making it the second tallest in Colombia after the 196-meter, 52-story "Torre Colpatria", in downtown Bogotá.

Built in the early 80's, the tower is built in the International Style. It currently houses a hotel and became a landmark building in the city, as it dominates the skyline.
