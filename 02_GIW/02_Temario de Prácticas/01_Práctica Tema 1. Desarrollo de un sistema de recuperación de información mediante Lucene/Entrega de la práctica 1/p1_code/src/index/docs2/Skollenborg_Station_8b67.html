Skollenborg Station (Norwegian: Skollenborg stasjon) is a railway station located at the village of Skollenborg in Kongsberg, Norway on the railway Sørlandsbanen. The station is served by local trains between Kongsberg via Oslo to Eidsvoll operated by Norges Statsbaner.

[edit] History

The station was opened in 1871 a branch line of Randsfjordbanen was opened between Hokksund and Kongsberg.
