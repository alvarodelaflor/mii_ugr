Tallong Midge Orchid
Scientific classification
Kingdom: Plantae

Family: Orchidaceae
Genus: Genoplesium
Species: G. plumosum
Binomial name
Genoplesium plumosum

The Tallong Midge Orchid (Genoplesium plumosum) is a tiny flower that grows only in the village of Tallong in the Southern Highlands of New South Wales. Its discovery brought the village to the attention of botanists and conservationists. This orchid is now a protected species.
