Al Saqr (Falcon; Arabic: الصقر) is an Yemeni football club based in Ta'izz, Yemen. The club was founded in 1969.

[edit] Achievements

  • Yemeni League: 1
    • Champion : 2006

[edit] Performance in AFC competitions

  • AFC Cup: 2 appearances
2007: Group Stage
