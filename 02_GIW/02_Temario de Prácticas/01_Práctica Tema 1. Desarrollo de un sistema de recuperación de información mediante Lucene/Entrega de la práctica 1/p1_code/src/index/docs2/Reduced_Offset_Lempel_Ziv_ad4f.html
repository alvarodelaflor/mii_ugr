Reduced Offset Lempel Ziv

From Wikipedia, the free encyclopedia

Reduced Offset Lempel-Ziv, (ROLZ) refers to variants of the LZ77 lossless data compression algorithms with an emphasis on improving throughput by efficient use of a table of contexts during compression.

One ROLZ implementation exists as part of the WinRK compression suite. The QUAD compressor includes a free implementation (licensed under the LGPL).
