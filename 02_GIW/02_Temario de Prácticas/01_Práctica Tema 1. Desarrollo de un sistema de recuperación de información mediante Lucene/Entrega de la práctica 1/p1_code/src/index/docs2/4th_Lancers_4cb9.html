4th Lancers

From Wikipedia, the free encyclopedia

The 4th Lancers was a part of the British Indian Army.

[edit] Origin

Raised 1838 the various changes and amalgamations are listed below.

[edit] History

  • 1838 Oudh Auxiliary Force Regiment
  • 1840 6th Irregular Cavalry
  • 1861 4th Regiment of Bengal Cavalry
  • 1900 4th Regiment of Bengal Lancers
  • 1903 4th Lancers
  • 1904 4th Cavalry
  • 1921 Amalgamated with 2nd Lancers (Gardner's Horse), to form 2nd Lancers (Gardner's Horse)[1]
