ISO 3166-2:2002-12-10

From Wikipedia, the free encyclopedia

ISO 3166-2:2002-12-10 is an ISO Newsletter (numbered 4) which was issued on December 10, 2002.

This Newsletter announced changes to ISO 3166-2. Affected codes were:

BI Burundi
CA Canada
EC Ecuador
ES Spain
ET Ethiopia
GE Georgia
ID Indonesia
IN India
KG Kyrgyzstan
KH Cambodia
KP North Korea
KZ Kazakstan
LA Laos
MD Moldova
MU Mauritius
RO Romania
SI Slovenia
TJ Tajikistan
TM Turkmenistan
TL East Timor
TW Taiwan
UZ Uzbekistan
VE Venezuela
YE Yemen
