"Popcorn Love/Jealous Girl" is a double A-side single by New Edition. Released in 1983,[1] the single reached #25 on the R&B chart.

[edit] Cover version

Another Bad Creation covered “Jealous Girl”[3] (in a slightly extended version)[4] on July 23, 1991, which also reached #25 on the R&B chart.[5]
