WLFN (1490 AM) is a radio station broadcasting a Adult Standards format. Licensed to La Crosse, Wisconsin, USA, the station serves the La Crosse area. The station is currently owned by Mississippi Valley Broadcasters, LLC and features programing from CNN Radio.[1]

[edit] History

The station went on the air as WLXR on 1983-09-07. on 1988-12-16, the station changed its call sign to the current WLFN.[2]
