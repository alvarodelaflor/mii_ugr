Moses Kamut

From Wikipedia, the free encyclopedia

Moses Kamut (born July 7, 1982) is a ni-Vanuatu sprinter who competed in the 100 metres at the 2004 Summer Olympics in Athens, Greece and the 2006 Commonwealth Games in Melbourne, Australia. He will be competing in the 2008 Summer Olympics in Beijing[1].
