2829 Bobhope

From Wikipedia, the free encyclopedia

2829 Bobhope is a small main belt asteroid, which was discovered by E. L. Johnson in 1948. It is named after American comedian Bob Hope.

Little is known about Bobhope, but it is known to measure 38.26 km in diameter, and to have a rotation period of 0.25 days (6 hours).
