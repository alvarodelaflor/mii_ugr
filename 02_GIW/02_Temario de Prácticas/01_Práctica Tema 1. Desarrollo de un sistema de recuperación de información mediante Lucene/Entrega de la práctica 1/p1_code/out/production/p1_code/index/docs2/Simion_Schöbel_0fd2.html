Simion Schöbel

From Wikipedia, the free encyclopedia

Olympic medal record
Men's Handball
Bronze 1972 Munich Team

Simion Schöbel (born February 22, 1950) is a former Romanian handball player who competed in the 1972 Summer Olympics.

In 1972 he won the bronze medal with the Romanian team. He played all six matches and scored two goals.
