Miss USA 1954

From Wikipedia, the free encyclopedia

Miss USA 1954
Presenters: Bob Russell
Venue: Long Beach Municipal Auditorium, Long Beach, California
Winner: Miriam Stevenson
Represented: South Carolina

The third edition of Miss USA was held on 1954, at Long Beach Municipal Auditorium, Long Beach, California. South Carolina, Miriam Stevenson, won the competition.

[edit] Results

  • Miss USA 1954: Miriam Stevenson, (South Carolina)
  • The runners-up were:
    • 1st Runner-up: Ellen Whitehead, (Virginia, was dethroned)
    • 2nd Runner-up: Renee Roy, (New York City)
    • 3rd Runner-up: Karin Hultman, (New York)
    • 4th Runner-up: Celeste Ravel, (Illinois)
