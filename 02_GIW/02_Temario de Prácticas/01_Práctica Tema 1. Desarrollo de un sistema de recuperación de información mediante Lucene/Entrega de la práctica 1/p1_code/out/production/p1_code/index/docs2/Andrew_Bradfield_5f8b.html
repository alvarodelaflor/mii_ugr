Andrew Bradfield

From Wikipedia, the free encyclopedia

Andrew "Andy" Bradfield (1966â€”September 21, 2001) was a computer programmer from New Zealand best known for his work on games for the Atari 8-bit series of computers. He created the classic games Hawkquest and Laser Hawk.

He died on September 21, 2001, aged 35, following a two-year battle with leukemia.
