Tokugawa Tsugutomo

From Wikipedia, the free encyclopedia

Tokugawa Tsugutomo (徳川継友, Tokugawa Tsugutomo?) (March 25, 1692-January 5, 1731) was a Japanese daimyo of the Edo period, who ruled the Owari Domain.
