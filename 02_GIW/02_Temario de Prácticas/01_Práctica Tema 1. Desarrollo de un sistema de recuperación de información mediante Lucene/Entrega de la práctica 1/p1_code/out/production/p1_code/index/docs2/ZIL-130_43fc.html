ZIL-130 is a Soviet/Russian truck designed and developed by ZIL, Moscow, Russia.

Replaced an older model ZIL-164. First prototypes - 1958. Since 1962 in serial production, since 1964 in mass production. One of the most popular cargo truck in USSR and Russia, in total ZIL made 3,380,000 up to 1994. Since 1995 production moved to the Urals automotor Zavod (UamZ) (renamed UamZ-43140)

ZIL-130 became the first ZIL truck painted in civil white and blue colors. Before this model virtually all ZIL trucks were painted in khaki military colors.
