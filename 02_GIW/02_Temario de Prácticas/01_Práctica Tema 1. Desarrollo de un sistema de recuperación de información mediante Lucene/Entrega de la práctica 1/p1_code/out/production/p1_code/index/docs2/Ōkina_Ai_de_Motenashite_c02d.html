"Ōkina Ai de Motenashite" (大きな愛でもてなして, Entertain Me With a Big Love?) is the third single from the Japanese pop group °C-ute, released on July 9, 2006 under an indies label.

This song was used in the anime Kirarin Revolution as the second ending song.
