Margaret (マーガレット, Margaret?) is a biweekly Japanese shōjo manga magazine published by Shueisha, primarily for girls from 11 to 15 years old, although some stories are read by adult women. It was first released as a weekly magazine in 1963.

When manga serialised in Margaret are collected into tankōbon collections, they are imprinted by Shueisha under the name "Margaret Comics".
