Titania (disambiguation)

From Wikipedia, the free encyclopedia

Titania may refer to:


  • Titania, the queen of the fairies in medieval folklore and a character from William Shakespeare's play, A Midsummer Night's Dream
  • Titania (DC Comics) is a DC Comics character
  • Titania (Marvel Comics), a Marvel Comics supervillain-ess
  • Titania (Grapplers), another Marvel Comics character
  • Titania (moon), a moon of the planet Uranus
  • Titania, a planet in the Star Fox series of video games
  • Titania, a Paladin in the game Fire Emblem: Path of Radiance
  • 593 Titania, an asteroid
  • The common name of titanium dioxide
  • Titania, the name of a titanium mine in Sokndal, Norway
  • Titania is the name of the main computer of the titular spaceship in the PC adventure game, Starship Titanic.
