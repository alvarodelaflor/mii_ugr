Ashuapmushuan River

From Wikipedia, the free encyclopedia

The Ashuapmushuan River runs between St-Felicien, La Dore and St-Methode and Normandin and St-Thomas in Canadian province of Quebec.

It is the location of Chute-à-l'Ours of Normandin camping camp.
