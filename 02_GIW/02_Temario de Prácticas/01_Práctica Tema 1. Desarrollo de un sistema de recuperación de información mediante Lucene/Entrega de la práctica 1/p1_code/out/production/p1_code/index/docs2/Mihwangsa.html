Mihwangsa

Korean name
Hangul 미황사
Hanja 美黃寺
Revised Romanization Mihwangsa
McCune-Reischauer Mihwangsa

Mihwangsa is a Buddhist temple in located at the southern-most point on the Korean peninsula, Haenam County, South Jeolla Province, South Korea.[1]Mihwangsa is a branch temple of Daeheungsa, the head temple of the 22nd district of the Jogye Order of Korean Buddhism.[2]
