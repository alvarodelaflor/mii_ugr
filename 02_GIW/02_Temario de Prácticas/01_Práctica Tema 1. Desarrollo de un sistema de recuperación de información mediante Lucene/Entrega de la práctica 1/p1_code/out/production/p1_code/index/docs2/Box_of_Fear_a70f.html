Box of Fear is the second album of Swedish progressive rock band Mister Kite. It marked a darker and heavier sound for Mister Kite.

  2. How Long

  5. Into Nothing

  7. State
  8. Do Your Worst
  9. The Hunger
  10. Box of Fear

  • Anton Johansson - bass
  • Thomas Djurfeldt - keyboards, guitars
  • Mats Bergentz - drums
