Henry Jenkins refers to world-making as "the process of designing a fictional universe that will sustain franchise development, one that is sufficiently detailed to enable many different stories to emerge but coherent enough so that each story feels like it fits with the others" (294).

[edit] Introduction

Some may get World-Making confused with World-building or Transmedia. This would be slightly off, the position were this term departs is in the world creation of multi-culture itself, not the media in which it spreads and grows. World-Making concentrates on the consumer side of franchising, not the different forms of media it spreads and connects.
