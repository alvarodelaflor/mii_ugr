Rhema FM

From Wikipedia, the free encyclopedia

Rhema FM is the name of a number of Christian radio stations formerly affiliated with United Christian Broadcasters (UCB) in Australia. Each is run and programmed locally, with predominantly local announcing. This is in contrast to the Vision Radio Network which is entirely produced by UCB from their Brisbane studios.

The name and logo of "Rhema FM" was derived from one of UCB's wholly owned networks in NZ that was then of the same name. See New Zealand's Rhema.

UCB no longer recognises affiliations with other radio stations because it has given up its role as a co-ordinating body to focus on operating as a broadcaster in its own right. Rhema FM stations may retain affiliation with each other by being members of Christian Media Australia (formerly the Association of Christian Broadcasters), which operates as the peak body for Christian media.
