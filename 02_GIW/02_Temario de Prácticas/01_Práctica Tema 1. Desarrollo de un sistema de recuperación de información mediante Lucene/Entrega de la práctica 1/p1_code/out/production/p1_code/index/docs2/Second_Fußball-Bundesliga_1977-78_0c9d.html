[edit] Promotion playoff

The final place in the Bundesliga was contested between the two runners-up in the North and South divisions.

  • 1. FC Nürnberg 1-0 Rot-Weiss Essen (June 1, 1978)
  • Rot-Weiss Essen 2-2 1. FC Nürnberg (June 9, 1978)

1. FC Nürnberg won 3-2 on aggregate and were promoted to the Bundesliga.
