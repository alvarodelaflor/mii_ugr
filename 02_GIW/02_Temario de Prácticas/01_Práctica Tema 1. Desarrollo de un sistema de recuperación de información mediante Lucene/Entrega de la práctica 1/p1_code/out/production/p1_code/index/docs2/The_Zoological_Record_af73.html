The Zoological Record (ZR) is the unofficial register of scientific names in zoology. It was started as a print publication in 1864 by the Zoological Society of London, as the The Record of Zoological Literature, and changed its name to the current one in 1870.

Since 200? it has been published by BIOSIS which is now owned by Thomson Scientific, a division of the Thomson Corporation.
