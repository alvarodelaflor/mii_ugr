No Sense

From Wikipedia, the free encyclopedia

For the song "No Sense" by Cat Power, see Moon Pix

No Sense is a Brazilian grindcore band, formed in São Paulo in 1990. They are one of few hardcore bands with a female vocalist. Similar bands include Necrose and Necrobutcher.
