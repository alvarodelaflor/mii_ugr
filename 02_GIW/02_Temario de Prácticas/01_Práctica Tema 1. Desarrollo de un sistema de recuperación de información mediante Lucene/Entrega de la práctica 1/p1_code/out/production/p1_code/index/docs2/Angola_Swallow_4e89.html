The Angola Swallow (Hirundo angolensis) is a species of bird in the Hirundinidae family. It is found in Angola, Burundi, Democratic Republic of the Congo, Gabon, Kenya, Malawi, Namibia, Rwanda, Tanzania, Uganda, and Zambia.
