Aldenham Country Park

From Wikipedia, the free encyclopedia

Aldenham Country Park is a 175-acre (0.71 kmÂ²) parkland and woodland, including a 65 acre reservoir, created in 1971 and now part of the Watling Chase Community Forest, located near Elstree, Hertfordshire in England. As well as offering community activities - walking, playgrounds, fishing, sailing - it features a Disney-backed Winnie-the-Pooh theme area. Since 1984 it has been an important centre for breeding rare livestock (one of seventeen Rare Breeds Survival Trust centres in the U.K.), including Aldenham Longhorn Cattle, Bagot goats, Berkshire and Tamworth pigs, Norfolk Horn and English Leicester sheep.
