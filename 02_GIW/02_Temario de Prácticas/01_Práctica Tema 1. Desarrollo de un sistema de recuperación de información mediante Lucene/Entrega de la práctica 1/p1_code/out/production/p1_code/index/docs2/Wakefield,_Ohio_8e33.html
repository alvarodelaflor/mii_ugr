Wakefield, Ohio

From Wikipedia, the free encyclopedia

Wakefield is an unincorporated community in southwestern Scioto Township, Pike County, Ohio, United States. Although it is unincorporated, it has a post office, with the ZIP code of 45687.[1] It lies along U.S. Route 23.
