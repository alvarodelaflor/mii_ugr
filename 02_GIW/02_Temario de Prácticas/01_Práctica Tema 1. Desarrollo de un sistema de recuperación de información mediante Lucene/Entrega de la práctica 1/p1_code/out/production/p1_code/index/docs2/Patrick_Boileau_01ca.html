Patrick Boileau

From Wikipedia, the free encyclopedia

Patrick Boileau (born February 22, 1975 in Montreal, Quebec) is a Canadian ice hockey defenceman.

Selected by the Washington Capitals in the 1993 NHL Entry Draft, he spent years in the Capitals system mainly with their AHL affiliate Portland Pirates until he signed with the Detroit Red Wings in 2002. In 2003 he signed with the Pittsburgh Penguins.
