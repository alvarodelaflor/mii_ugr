Attert (Atert in Luxembourgish, Ater in Walloon) is a municipality of Belgium. It lies in the country's Walloon Region and Luxembourg Province.

On 1 January 2007 the municipality, which covers 70.94 km², had 4,802 inhabitants, giving a population density of 67.7 inhabitants per km².

The municipality falls within the Luxembourgish-speaking linguistic area and several street name signs are bilingual.
