Thomas Arnold (bobsleigh)

From Wikipedia, the free encyclopedia

Olympic medal record
Bobsleigh
Silver 1924 Chamonix Four-man

Thomas Alfred Arnold (born 1901, date of death unknown) was a British bobsledder who competed during the early 1920s. He won a silver medal in the four-man event at the 1924 Winter Olympics in Chamonix.
