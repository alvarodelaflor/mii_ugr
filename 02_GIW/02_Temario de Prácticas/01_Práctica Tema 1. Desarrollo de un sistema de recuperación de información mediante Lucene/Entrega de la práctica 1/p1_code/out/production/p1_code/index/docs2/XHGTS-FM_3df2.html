XHGTS-FM 107.3 (branded as Digital 107.3) is an International Contemporary Music format FM radio station that serves the Laredo, Texas (USA) / Nuevo Laredo, Tamaulipas (Mexico) border area. They play mainly Spanish language pop and rock hits.
