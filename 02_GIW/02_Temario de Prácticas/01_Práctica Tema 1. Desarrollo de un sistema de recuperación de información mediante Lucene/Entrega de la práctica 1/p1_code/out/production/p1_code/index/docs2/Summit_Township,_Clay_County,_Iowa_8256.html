Summit Township is a township in Clay County, Iowa, USA. As of the 2000 census, its population was 452.

[edit] Geography

Summit Township covers an area of 35.3 square miles and contains one incorporated settlement, Fostoria. According to the USGS, it contains one cemetery, Evergreen.
