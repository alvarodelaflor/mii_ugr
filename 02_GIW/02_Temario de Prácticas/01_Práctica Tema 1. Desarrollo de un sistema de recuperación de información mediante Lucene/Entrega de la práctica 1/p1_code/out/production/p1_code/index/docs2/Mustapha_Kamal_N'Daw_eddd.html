Mustapha Kamal N'Daw (born: 15 November 1981) is a football player from Gambia. The midfielder currently plays for Veria FC in Veria. He has formerly played for AEK Larnaca, B68 Toftir, K.A.A. Gent and B71 Sandur and Doxa Katokopia.
