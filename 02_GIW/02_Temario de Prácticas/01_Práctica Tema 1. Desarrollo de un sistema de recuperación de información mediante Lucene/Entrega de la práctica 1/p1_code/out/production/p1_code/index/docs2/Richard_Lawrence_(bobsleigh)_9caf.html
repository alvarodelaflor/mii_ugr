Richard W. Lawrence (July 22, 1906 â€“ June 1974) was an American bobsledder who competed in the 1930s. He won the bronze medal in the two-man event at the 1936 Winter Olympics in Garmisch-Partenkirchen and finished sixth in the four-man event at those same games.

He died in Rochester, New York.
