Roberto D'Aversa (born 12 August 1975 in Stuttgart, Germany) is an Italian footballer, who plays as a midfielder for F.C. Messina Peloro of Serie A.

On January 28, 2007 he played his first Serie A match for Messina against Ascoli.[1]
