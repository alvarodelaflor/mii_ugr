The Best of Joy Division is a compilation album of material from Joy Division. It was released March 24, 2008 (2008-03-24) and the UK version includes The Complete BBC Recordings as a bonus disc. The US release is a single disc. The Best of Joy Division reached #97 in April 2008 in Australia, which marks only their second appearance on that chart after Substance reached #53 in 1988.
