Beyond the Black Hole is a Man or Astro-man? compilation featuring tracks that originally appeared on 7" EPs. It was released in 2001 on Estrus Records. This release features many of the same songs that were featured on the 1997 Australia-only release What Remains Inside a Black Hole.
