Vilda Mánnu is the second album by Eternal Tears of Sorrow. It contains the band's first guest vocals (by Heli Luokkala, known from Nicepappi). It is also the last EToS album with their original line-up as one of the co-founders of the band, Olli-Pekka Törrö, left the band a few months after the release of this album.

The title of the album (as well as the name of the title song), Vilda Mánnu, is Northern Sami language. It means "Wild Moon" in English. There is also another Sami song title on the album, Goashem, which (in the form "goas'kem") is Old Sami language for "an eagle" [1]
