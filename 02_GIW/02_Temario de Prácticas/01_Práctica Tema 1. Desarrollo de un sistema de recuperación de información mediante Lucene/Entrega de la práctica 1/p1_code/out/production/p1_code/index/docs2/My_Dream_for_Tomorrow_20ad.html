My Dream for Tomorrow

From Wikipedia, the free encyclopedia

"My Dream for Tomorrow" is a song from the musical Over Here!, which opened on Broadway at the Shubert Theatre (Broadway) on March 6, 1974. The song was written by Robert B. Sherman and Richard M. Sherman. It is one of the Sherman Brothers' counterpoint songs which are well known for juxtaposition of complimentary melody lines. Generally, these songwriters choose to use the counterpoint device when two characters are coming from opposing points of view.
