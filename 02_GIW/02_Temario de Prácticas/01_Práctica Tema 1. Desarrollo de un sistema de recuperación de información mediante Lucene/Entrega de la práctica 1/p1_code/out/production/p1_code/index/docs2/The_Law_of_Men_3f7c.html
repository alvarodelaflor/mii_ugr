The Law of Men is a 1919 silent drama film directed by Fred Niblo. [1]

[edit] Cast

  • Enid Bennett - Laura Dayne
  • Niles Welch - Denis Connors
  • Andrew Robson - Benton Wade
  • Dorcas Matthews - Mildred Wade (as Dorcas Mathews)

  • Frankie Lee - Child
