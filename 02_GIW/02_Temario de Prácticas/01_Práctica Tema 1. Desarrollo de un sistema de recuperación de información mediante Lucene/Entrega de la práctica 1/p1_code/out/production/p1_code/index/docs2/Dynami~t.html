Dynami:t

From Wikipedia, the free encyclopedia

Dynami:t is an energy drink produced and distributed by A. Le Coq from Tartu, Estonia.

The caffeine content of each beverage is 23.8mg per 100ml.

The drinks are usually sold in 330ml cans with a 0.50 EEK deposit.
