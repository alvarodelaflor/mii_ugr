The Machine God Laughs is an anthology of three science fiction short stories anonymously edited by William L. Crawford. It was published as Griffin Publishing Company in 1949 in an edition of 1,200 copies. The stories originally appeared in the magazine Fantasy Book.

[edit] Contents

  • "The Machine-God Laughs", by Festus Pragnell
  • "Star of the Undead", by Paul Dennis Lavond (pseudonym for Robert A. W. Lowndes, Frederik Pohl & Joseph Harold Dockweiler)
  • "Crusader", by Basil Wells
